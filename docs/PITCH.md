# Ластівка — Пітч (наратив + демо-сценарій)

## Один меседж (вивчити напамʼять)
> **«Сьогодні сигнали про дитину в ризику розкидані по 20+ держреєстрах, які не бачать одне
> одного. Злити їх — незаконно й небезпечно. Ластівка дає бачити дитину ЦІЛІСНО, не зливаючи
> бази: реєстри порівнюються за зашифрованими відбитками, найчутливіше — лише сигнал «є/нема».»**

Три солюшени — це не три продукти, а **відповіді на три питання про ОДНУ систему**:
| Пілар | Питання журі | Наша відповідь |
|---|---|---|
| 1. Безпека даних | «А це законно/безпечно?» | PSI/PPRL + стіни — бачимо, не зливаючи |
| 2. Система ризиків | «А воно реально працює?» | Пояснюваний скоринг + аудит «ризик vs хвороба», F1≈0.87 |
| 3. Ролі + Естонія | «Хто це юзає і як масштабувати?» | Рольовий доступ + крос-кордон UA↔EE для дітей-біженців |

## Структура слайдів (7)

**S1 — Проблема (емоція + цифра).** Дитина випадає крізь систему: не в школі + пропущена
медицина + ДН удома — і ніхто не бачить повної картини, бо кожен реєстр окремо. *«Не можна
просто злити бази — це масове стеження + порушення медтаємниці/таємниці слідства».*

**S2 — Рішення (один кадр).** Ластівка = privacy-preserving шар над реєстрами. Три пілари
(таблиця вище). Тут показати **діаграму потоку даних** (`docs/img/dataflow.png`).

**S3 — Пілар 1: Безпека (як бачити, не зливаючи).** PSI (`HMAC(УНЗР)` — токен, не дані),
PPRL (Bloom-відбитки імен), **стіни** (ЕСОЗ/ЄРДР — лише «є/нема», зміст не читається). Меседж:
*«Найчутливіше не бачить ніхто, навіть поліція — лише лікар/суд»*.

**S4 — Пілар 2: Ризики (що виявляємо + чому довіряти).** Черга T0/T1/T2 з **поясненням кожного
рішення** (не чорна скриня). Кульмінація довіри — **аудит**: *«Ми перевірили, що система НЕ
позначає дитину, яка просто захворіла, як жертву насильства»* (диференціал ризик-vs-хвороба +
запобіжники: оракул не дав заглушити реальний ризик). F1≈0.87. Воєнні порушення (W10 мілітаризація,
W4 поранення) — релевантно Україні.

**S5 — Пілар 3: Ролі + Естонія.** Кожен актор бачить лише легальне: ССД — своя громада з ПІБ,
поліція — вищий допуск (ЄРДР), регіонал — агрегати без ПІБ. Крос-кордон: дитина-біженець у
Естонії — зв'язок UA↔EE через PPRL (перетинає лише результат збігу, GDPR гл.V). *«Готові до
співпраці: Трембіта ↔ X-Road Естонії вже сумісні»*.

**S6 — Чесність + прод.** Що реальне зараз (крипта приватності, детекція, аудит — на синтетиці,
0 реального ПІБ) vs прод (федеративні реєстри через Трембіту, MPC/TEE, Дія.Підпис, журнал
доступу). **Людина в циклі**: рішення ухвалює Комісія, система — підтримка, не вирок.

**S7 — Impact + ask.** Раніше виявлення → менше дітей крізь тріщини. Наступний крок: пілот на
1 громаді через Трембіту + DPIA. (Що просимо: доступ/партнерство/менторство).

## Демо-сценарій (3 хв, по кроках)
1. **Вхід як ССД** (Андрій, Харків) → **черга**: діти за терміновістю, T0 згори. *«Система вже
   розставила пріоритети й пояснює, чому».*
2. **Відкрити дитину T0** → профіль, зібраний з реєстрів (НЕ злитих): силоси, **сімейний граф**,
   **«Слід в Естонії»**, **медфакти**. Показати **«1 захищене джерело — медтаємниця»** = стіна тримається.
3. **Перемкнути на Регіонала** → ті самі дані, але **без ПІБ** (фізично вирізано) → сегрегація.
4. **Діаграма потоку** (S2) — *«ось як дані течуть: відбитки, не бази»*.
5. **Момент-аудит**: відкрити `out/audit/report.md` або сказати: *«ми запустили флот агентів, що
   інжектять кейси й шукають, де система помиляється — і виправили, не зробивши її сліпою до ризику»*.

## 5 диференціаторів (чому ми, а не інші)
1. **Не злиття баз, а privacy-fabric** — легально + безпечно (PSI/PPRL/стіни).
2. **Пояснюваність** — кожне рішення з причиною, не чорна скриня.
3. **Аудит/харнес** — ми реально тестували «ризик vs хвороба» (більшість команд цього не роблять).
4. **Український контекст** — воєнні порушення + співпраця з Естонією для дітей-біженців.
5. **Людина в циклі** — підтримка рішень, не автоматичний вирок.

## Анти-питання (що спитає журі — і відповідь)
- *«Це не стеження?»* → Ні: бачимо лише результат збігу, зміст walled-реєстрів не читається; рольова мінімізація; людина ухвалює.
- *«Дані реальні?»* → Синтетичні, 0 ПІБ; крипта приватності справжня й виміряна; прод — федеративно через Трембіту.
- *«А якщо хибно звинуватить?»* → Це підтримка, не вирок; кожне рішення пояснюване; аудит ловить хибні спрацювання; запобіжники проти FN.
- *«Чому Естонія?»* → Найбільше українських дітей-біженців + X-Road сумісний з Трембітою → реальний пілот співпраці.
