# 🕊️ Ластівка — фронтенд

Веб-інтерфейс державного рівня для системи раннього виявлення порушень прав дитини.
Чистий, серйозний, доступний GovTech-дизайн поверх FastAPI-бекенду.

**Стек:** Next.js 16 (App Router) · TypeScript · Tailwind CSS v4 · кастомні SVG-графіки (без важких залежностей).

## Запуск

```bash
cd frontend
npm install
npm run dev          # http://localhost:3000
```

За замовчуванням працює на **вбудованих демо-даних** — бекенд не потрібен.
Щоб підключити реальний API, створи `.env.local`:

```
NEXT_PUBLIC_API_BASE=http://localhost:8000
```

і підніми бекенд (`uvicorn app.api:app --port 8000`). За помилки мережі фронт
автоматично відкочується на моки.

## Сторінки

| Маршрут | Стовп | Що показує |
|---|---|---|
| `/` | — | Лендінг: проблема, завдання, як працює, для кого (за слайдами) |
| `/dashboard` | ④ | Управлінська панель: KPI, графіки, якість моделі |
| `/queue` | ③ | Черга реагування: ранжовані кейси з **поясненням** score |
| `/profile` | ① | Профіль дитини: силоси, таймлайн, графік зламу (change-point) |
| `/privacy` | ② | Приватність (PPRL) і якість виявлення (precision/recall) |

## Структура

```
src/
  app/
    page.tsx              # лендінг
    (platform)/           # робочі екрани (спільний AppShell із sidebar)
      dashboard|queue|profile|privacy/page.tsx
  components/
    landing/  layout/  ui/  charts/  queue/  profile/
  lib/
    types.ts              # типи — дзеркало контракту backend
    registries.ts         # метадані реєстрів і порушень
    mock.ts               # детерміновані демо-дані (12 hero-кейсів + популяція)
    api.ts                # шар даних: API ↔ моки
    format.ts             # форматери (укр. локаль)
```

## Дизайн

- **Кольори терміновості** наскрізні: 🔴 T0 · 🟠 T1 · 🟡 T2.
- Один бренд-акцент (sky-teal «ластівка»), графітовий текст, світлий фон.
- Доступність: контраст, `prefers-reduced-motion`, фокус-стилі, семантика.
- Тон UI: **decision support, не вирок** — система пріоритезує й пояснює.
